Controlling this Graphics Canvas:
  • There are three modes:
    1. Camera mode: controlled by w (up), a (left), s (down), d (right), and you can rotate the camera with f.
    2. Ball mode: ALT + a to activate. This allows you to control the ball's x and z movement with the arrow keys. You can move up by pressing c. Hitting ALT + c turns on gravity, but there is a bug where you cannot move with the arrow keys or else your ball matrix will go off the game scene.
    3. Hamster pan mode: ALT + x to activate. Pans the camera on the hamster.
  • To check camera shading, use ALT + n.
  • Requirements fulfilled:
    1. Two-level hierarchical object: sunflower
    2. lookAt: ball and hamster
    3. Custom shapes: sunflower head, cheese
    4. Flat shading: cheese
    5. Texture: ball
